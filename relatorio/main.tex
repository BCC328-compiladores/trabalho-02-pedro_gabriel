\begin{abstract}
Este trabalho apresenta o desenvolvimento das etapas de Análise semântica e Interpretador de um compilador desenvolvido na linguagem Haskell, para a linguagem SL, e consiste na segunda etapa do trabalho prático de Construção de Compiladores. Este documento descreve o processo de implementação desta etapa do compilador, juntamente com a divisão do trabalho entre os participantes e os recursos de LLM utilizados.
\end{abstract}
